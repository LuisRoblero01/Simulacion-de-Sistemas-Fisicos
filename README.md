# Simulation of Physical Systems

Coursework projects and assignments from the **Simulation of Physical Systems**
course: numerical methods and scientific-computing exercises implemented in
Python.

## Contents

- **T1 — Polynomial evaluation & plotting** — implements and compares **four
  different algorithms** for evaluating polynomials (including a naive approach
  and Horner-style schemes), and visualizes the results.

More assignments and numerical-methods exercises are added as the course
progresses.

## Topics

Numerical methods · algorithm comparison · scientific computing · data
visualization.

## Tech stack

Python · NumPy · Matplotlib
